## Overview

An AI-powered chatbot built to help users query and understand UAP release files. Uses retrieval-augmented generation to surface accurate answers from release documentation instead of relying on generic LLM responses.

## Key Features

- Document-grounded Q&A over release file corpora
- Spring Boot backend with Spring AI integration
- Deployed live on Render for interactive demos
